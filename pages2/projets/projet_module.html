  1. Créer un module joueraucartes qui contiendra :
    - Les fonctions :
    1. Crée un jeu de 32 cartes sous la forme d’un tableau de tuples (nom,valeur)
    2. Crée un jeu de 52 cartes sous la forme d’un tableau de tuples (nom,valeur)
    3. Affiche le nom d’une carte (exemple 8 Coeur, Valet Trèfle,...)
    4. Mélange un jeu de cartes (interdit d'utiliser shuffle !)
    5. Tire la 1ere carte d’un paquet et l’enlève du paquet
    6. Compare 2 cartes et retourne la meilleure valeur si possible
    - Les constantes :
    1. figure32 = ["7","8","9","10","valet","dame","roi","as"]
    2. figure52= ["2",…..................."as"].
    3. Couleur= ["coeur","carreau","pique","trefle"]
  2. Dans un programme principal, qui utilisera le module joueraucartes, simuler le jeu de batailles (on affichera chaque coup joué et le score des 2 joueurs) on poura simuler un a autre jeu de cartes ( si validé par le professeur)
