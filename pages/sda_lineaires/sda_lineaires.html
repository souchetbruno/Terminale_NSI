Structures de données linéaires

    Nous allons dans ce cours décrire des représentations des structures de données telles que les piles, les files et les listes chaînées.
    Ce sont des structures de données à une dimension permettant de stocker de manière linéaire une suite finie de données. On les dénomme aussi structures de données linéaires.

  1. Les Piles :
    1. Cours : Définition, exemples, exercices en débranché
    2. Notebook : Implémentations d'une pile en python
    3. TP : Manipulation des piles
    4. Notebook : 2 exercices sur l'application des piles: Bon parenthésage d'une expression et Notation Polonaise Inversée
    5. Exercice sur la résolution d'un labyrinthe
  2. Les Files :
    1. Cours : Définition, exercices
    2. Notebook : Implémentations d'une file en python + qqs exercices
    3. TP : Croisement routier
  3. Les Listes Chaînées :
    1. Cours : Définition, listes en Python, listes chaînées
    2. Implémentation d'une liste chainée
    3. Le lièvre et la tortue : detection de boucles dans une liste chainée
